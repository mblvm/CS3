# CS3 · Browser Strike

Браузерный FPS-шутер в стиле Counter-Strike 2 на Three.js: раунды с закладкой бомбы, экономика, боты двух команд.

## Запуск

```bash
npm install
npm run dev
```

Откройте http://localhost:5173, выберите сложность и нажмите «Играть».

## Игровой цикл

- **Раунды**: вы начинаете за террористов. Возьмите бомбу (носитель — случайный игрок T, отмечен на радаре), доставьте на точку **A** или **B** и удерживайте **E** для установки. Таймер бомбы — **40 секунд**; спецназ разминирует удержанием **E** (6 сек). Раунд также выигрывается уничтожением всей команды противника или (для CT) по истечении времени раунда (1:55).
- **Экономика**: деньги за фраги ($300, нож $1500), победу ($3250–3500) и поражение (компенсация растёт с серией, $1400–3400). Закупка на **B** в начале раунда: AK-47 $2700, AWP $4750, броня $650. Купленное оружие теряется при смерти, но сохраняется, если выжили.
- **Матч**: до **13** выигранных раундов, **смена сторон после 12 раундов** (деньги и арсенал сбрасываются). Погибнув, вы наблюдаете до конца раунда — возрождение в следующем.

## HUD и управление

- Прицел с динамическим разбросом, HP/броня, деньги, патроны, таймер раунда/бомбы, счёт по раундам, killfeed, **радар** (миникарта: союзники, замеченные враги, бомба), табло на **Tab**.
- **WASD** — движение, **Пробел** — прыжок (распрыжка), **Ctrl/C** — присед, **Shift** — тихий шаг, **1–4** — оружие, **R** — перезарядка, **B** — закупка, **E** — установка/разминирование, **Tab** — табло, **Esc** — меню.
- Главное меню: старт, сложность ботов, чувствительность мыши.

## Технически

- **Движок**: Three.js + Pointer Lock API, процедурные canvas-текстуры, синтезированные звуки (WebAudio) — без внешних ассетов.
- **Физика**: гравитация, коллизии AABB, **распрыжка** (air-strafe по механике Quake/Source).
- **Карта** в духе de_dust2: T-спавн (юг), CT-спавн (север), лонг, мид, тоннели, точки **A** и **B**.
- **Оружие**: нож, USP-S, AK-47 (паттерн спрея), AWP (зум на ПКМ). Разброс от движения, хедшоты ×4.
- **Боты**: две команды со своими задачами — T несут и ставят бомбу, CT защищают точки и разминируют; патруль по waypoint-графу (BFS), бой бот-против-бота, укрытия при низком HP, подбор брошенной бомбы. Три уровня сложности (размер команд, реакция, меткость, урон).
- **Оптимизация**: проверки зрения ботов на 10 Гц со сдвигом фаз, отбраковка коллайдеров по описанной сфере в raycast, запечённый фон радара, переиспользование временных векторов.

## Структура

```
src/
  main.js      — обвязка: рендер, ввод, меню, цикл кадра
  gamestate.js — раунды, бомба, экономика, счёт, смена сторон
  player.js    — физика игрока (движение, распрыжка, присед, броня)
  weapons.js   — оружие, разброс, отдача, hitscan, закупка
  bots.js      — ИИ двух команд (задачи раунда, бой, укрытия)
  map.js       — геометрия карты, коллайдеры, waypoint-граф, сайты, спавны
  radar.js     — миникарта
  hud.js       — интерфейс (таймер, деньги, killfeed, табло, закупка)
  textures.js  — процедурные текстуры
  effects.js   — трассеры, вспышки, взрыв, частицы
  audio.js     — синтезированные звуки
  utils.js     — коллизии AABB, лучи, математика
```
