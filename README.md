# CS3 · Browser Strike

Браузерный FPS-шутер в стиле Counter-Strike 2 на Three.js.

## Запуск

```bash
npm install
npm run dev
```

Откройте http://localhost:5173, выберите сложность и нажмите «Играть».

## Возможности

- **Движок**: Three.js + Pointer Lock API, процедурные canvas-текстуры, синтезированные звуки (WebAudio) — без внешних ассетов.
- **Управление**: WASD, мышь, Пробел — прыжок, Ctrl/C — присед, Shift — тихий шаг, 1–4 — оружие, R — перезарядка, Tab — счёт, Esc — меню.
- **Физика**: гравитация, коллизии со стенами и ящиками (можно запрыгивать на низкие ящики), **распрыжка** — держите Пробел и стрейфьтесь мышью в воздухе (air-strafe по механике Quake/Source), индикатор скорости слева внизу.
- **Карта** в духе de_dust2: T-спавн (юг), CT-спавн (север), три линии — лонг, мид, тоннели — и две точки закладки **A** и **B** с укрытиями из ящиков.
- **Оружие**: нож, USP-S, AK-47 (паттерн спрея — вверх, затем зигзаг), AWP (прицел на ПКМ). Разброс зависит от движения/приседа/стрельбы, хедшоты ×4, перезарядка, запас патронов.
- **Боты**: патрулирование по waypoint-графу с поиском пути (BFS), обнаружение игрока по прямой видимости и полю зрения, реакция на выстрелы и урон, стрельба очередями с разбросом, стрейф в бою, отход в укрытие при низком HP. Три уровня сложности (меняются реакция, меткость, урон, дальность обзора и число ботов).

## Структура

```
src/
  main.js     — цикл игры, ввод, меню
  player.js   — физика игрока (движение, распрыжка, присед)
  weapons.js  — оружие, разброс, отдача, hitscan
  bots.js     — ИИ ботов (патруль/бой/поиск/укрытие)
  map.js      — геометрия карты, коллайдеры, waypoint-граф
  textures.js — процедурные текстуры
  effects.js  — трассеры, вспышки, частицы
  audio.js    — синтезированные звуки
  hud.js      — интерфейс
  utils.js    — коллизии AABB, лучи, математика
```
